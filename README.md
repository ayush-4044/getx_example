# 🛍️ TrendStore - GetX E-Commerce App

A high-performance Flutter application built with **GetX** for state management, demonstrating professional project architecture and reactive programming.

## ✨ Key Features
- **Smart Splash Screen**: Automatic session handling (Login vs Home).
- **Flexible Login**: User-friendly authentication with email validation.
- **Persistent Settings**: Dark/Light mode preference saved using `GetStorage`.
- **Reactive Cart**: Real-time counter updates using GetX `Obx`.
- **Premium UI**: Modern gradient design with Google Fonts.

## 🛠️ Tech Stack
- **Framework:** Flutter    
- **State Management:** GetX
- **Local Storage:** GetStorage
- **Typography:** Google Fonts (Poppins)

## 📂 Project Structure
- `lib/controller`: Business logic and state handling.
- `lib/Screens`: Clean UI components (Splash, Login, Home).
- `lib/service`: Authentication and session management.
